无机化学：配合物与晶体场理论

覆盖配合物命名、异构现象、晶体场分裂能、光谱化学序列、磁性判断及姜-泰勒效应。

概述

配合物化学是无机化学最具挑战性的章节。本笔记覆盖配合物的命名、异构现象、晶体场理论、磁性判断和反应动力学。

学习目标：

  • 掌握配合物的系统命名和异构体判断
  • 理解晶体场分裂能和光谱化学序列
  • 能够判断配合物的磁性和自旋状态

第一章：配合物基础

基本概念

中心原子/离子：通常是过渡金属离子（d 轨道未满）
配体：与中心原子结合的分子或离子
配位数：中心原子周围配体的数量（常见 2, 4, 6）
配位原子：配体中直接与中心原子成键的原子

常见配体

类型 例子 配位原子 齿数
单齿 NH₃, H₂O, Cl⁻, CN⁻ N, O, Cl, C 1
双齿 en（乙二胺）, ox²⁻（草酸根） N, O 2
多齿 EDTA⁴⁻ N, O 6

命名规则

1. 配体名称在前，中心原子在后
2. 配体按字母顺序排列（不看前缀数字）
3. 配体数量用 二、三、四... 表示
4. 中心原子氧化态用罗马数字标注

示例：
  [Co(NH₃)₆]Cl₃ → 氯化六氨合钴(III)
  K₄[Fe(CN)₆] → 六氰合铁(II)酸钾
  [Pt(NH₃)₂Cl₂] → 二氯·二氨合铂(II)

第二章：异构现象

几何异构

平面正方形（配位数 4）：
  [Pt(NH₃)₂Cl₂]
  
  顺式 (cis)：相同配体相邻→ 有偶极矩
  反式 (trans)：相同配体相对→ 无偶极矩

八面体（配位数 6）：
  [Co(NH₃)₄Cl₂]⁺
  
  顺式：两个 Cl⁻ 占据相邻顶点（90°）
  反式：两个 Cl⁻ 占据相对顶点（180°）

光学异构

[M(AA)₃] 型（AA 为双齿配体）：
  如 [Co(en)₃]³⁺
  
  Δ 异构体（右旋）：右手螺旋
  Λ 异构体（左旋）：左手螺旋
  
  二者互为对映异构体，旋光度大小相等、方向相反

其他异构类型

类型 说明 例子
电离异构 内外界离子交换 [Co(NH₃)₅Br]SO₄ vs [Co(NH₃)₅SO₄]Br
水合异构 结晶水位置不同 [Cr(H₂O)₆]Cl₃ vs [Cr(H₂O)₅Cl]Cl₂·H₂O
配位异构 配体连接原子不同 -NO₂（硝基）vs -ONO（亚硝酸根）

第三章：晶体场理论 (CFT)

核心假设

  1. 配体被视为点电荷（或点偶极）
  2. 中心金属的 d 轨道与配体的静电相互作用导致 d 轨道能级分裂
  3. 分裂方式取决于配位几何构型

八面体场

d 轨道分裂：
       ↑  eg (d_x²-y², d_z²)
  Δo（分裂能）
       ↓  t₂g (d_xy, d_xz, d_yz)

Δo 的影响因素：
1. 金属离子：氧化态越高，Δo 越大
2. 配体：光谱化学序列
3. 周期数：同族向下，Δo 增大（5d > 4d > 3d）

光谱化学序列

I⁻ < Br⁻ < S²⁻ < SCN⁻ < Cl⁻ < NO₃⁻ < F⁻ < OH⁻ < C₂O₄²⁻ < H₂O
< NCS⁻ < NH₃ < en < NO₂⁻ < CN⁻ < CO

弱场配体（左侧）→ 小 Δo → 高自旋
强场配体（右侧）→ 大 Δo → 低自旋

晶体场稳定化能 (CFSE)

CFSE = (-0.4n_t₂g + 0.6n_eg) × Δo（八面体场）

d⁴（八面体）：
  高自旋：t₂g³ eg¹ → CFSE = -0.6Δo
  低自旋：t₂g⁴ eg⁰ → CFSE = -1.6Δo + P（P = 电子成对能）

第四章：磁性

磁矩计算（仅自旋公式）

μ_s = √[n(n+2)]  BM（玻尔磁子）
n = 未成对电子数

d⁶（八面体）：
  高自旋（t₂g⁴ eg²）：n=4 → μ_s = √24 ≈ 4.90 BM
  低自旋（t₂g⁶ eg⁰）：n=0 → μ_s = 0（抗磁性）

磁性判断流程

1. 确定中心金属的 d 电子数
2. 确定配体场强弱（光谱化学序列）
3. 确定高/低自旋态 → 计算未成对电子数
4. 用公式计算磁矩
5. 对比实验值验证

第五章：姜-泰勒效应 (Jahn-Teller)

定理：非线形分子在简并电子基态下不稳定，
     会发生几何畸变以消除简并

八面体配合物中：
  高自旋 d⁴（t₂g³ eg¹）→ eg 轨道简并 → 拉长畸变
  d⁹（t₂g⁶ eg³）→ eg 轨道有一个空穴 → 拉长畸变
  Cu²⁺ 配合物几乎都发生 JT 畸变！

高频考点

  1. 配合物命名（中英文互译）
  2. 判断几何异构体数目（顺反异构）
  3. 用 CFT 解释配合物颜色（d-d 跃迁吸收可见光）
  4. 计算 CFSE 和磁矩
  5. 预测 JT 畸变的发生
  6. 光谱化学序列的应用
