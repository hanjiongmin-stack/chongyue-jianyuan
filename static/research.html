开源贡献路线图

五步完成你的第一个 Pull Request

1

注册 GitHub

创建账号，完善个人资料，了解 Star / Fork / Watch

2

找到项目

从 Good First Issue 或下方推荐项目中选一个

3

Fork + 开发

Fork 到自己的仓库，clone 到本地，创建分支

4

提交 PR

Push 到 Fork，发起 Pull Request，等待 Review

5

Merge 🎉

根据意见修改，代码合并——你已是开源贡献者
