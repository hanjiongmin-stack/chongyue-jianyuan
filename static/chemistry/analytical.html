分析化学：光谱与色谱方法精讲

四大光谱（UV-Vis/IR/NMR/MS）解析、HPLC/GC 色谱优化及滴定分析完整方法论。

概述

分析化学的核心任务是"定性"和"定量"。本笔记聚焦四大光谱（UV-Vis、IR、NMR、MS）、色谱方法（HPLC、GC）和滴定分析。

学习目标：

  • 掌握四大光谱的原理和谱图解析
  • 理解色谱分离的基本原理和优化策略
  • 熟练应用滴定分析中的计算方法

第一章：紫外-可见光谱 (UV-Vis)

本章目标

  • 理解Beer-Lambert定律
  • 掌握电子跃迁类型
  • 区分发色团与助色团
  • 能解析UV-Vis谱图

基本原理

分子吸收 UV-Vis 光 → 电子从基态跃迁到激发态

Beer-Lambert 定律：
  A = ε·b·c
  
  A：吸光度（Absorbance）
  ε：摩尔吸光系数（L·mol⁻¹·cm⁻¹）
  b：光程（cm）
  c：浓度（mol/L）

电子跃迁类型

跃迁类型 波长范围 ε 范围 例子
σ→σ* < 200 nm 大 烷烃
n→σ* 150-250 nm 中 含 O/N/S/卤素
π→π* 200-700 nm 很大 烯烃、芳香族
n→π* 250-600 nm 小 羰基化合物

发色团与助色团

发色团：具有 π→π* 或 n→π* 跃迁的基团
  C=C, C=O, N=N, 苯环

助色团：含有孤对电子的基团，与发色团相连时
        使吸收峰红移并增强
  -OH, -NH₂, -Cl

第二章：红外光谱 (IR) ⭐⭐⭐⭐⭐

本章目标

  • 掌握Hooke定律近似
  • 记忆特征官能团吸收区
  • 能根据IR判断官能团
  • 理解羰基化合物的IR特征

基本原理

分子振动能级跃迁吸收红外光
振动频率（Hooke 定律近似）：
  ν = (1/2πc)·√(k/μ)

  k：力常数（键能相关）
  μ：折合质量 = m₁m₂/(m₁+m₂)

特征吸收区

波数 (cm⁻¹) 官能团 振动类型
3200-3600 O-H（醇/酚） 伸缩振动
3300-3500 N-H（胺/酰胺） 伸缩振动
3000-3100 =C-H（烯/芳） 伸缩振动
2800-3000 -C-H（烷烃） 伸缩振动
1680-1750 C=O（羰基） 伸缩振动
1600-1680 C=C（烯/芳） 伸缩振动
1000-1300 C-O（醇/醚/酯） 伸缩振动

羰基化合物的 IR 特征

酰氯：~1800 cm⁻¹（最高波数）
酸酐：~1760 + ~1820 cm⁻¹（双峰）
酯：~1735 cm⁻¹
醛：~1725 cm⁻¹
酮：~1715 cm⁻¹
羧酸：~1710 cm⁻¹
酰胺：~1650 cm⁻¹（最低波数）

第三章：核磁共振 (NMR) ⭐⭐⭐⭐⭐

本章目标

  • 掌握化学位移概念
  • 理解n+1裂分规则
  • 能根据¹H NMR推断结构
  • 理解磁各向异性效应

¹H NMR 基础

化学位移 δ（ppm）：
  δ = (ν_sample - ν_TMS) / ν_spectrometer × 10⁶

影响化学位移的因素：
1. 电负性：相邻原子电负性越大，δ 越大（去屏蔽）
2. 磁各向异性：苯环（δ 7-8）、醛（δ 9-10）、炔（δ 2-3）
3. 氢键：使 δ 增大

¹H NMR 常见化学位移

质子类型 δ (ppm)
TMS 0
R-CH₃ 0.7-1.3
R-CH₂-R 1.2-1.4
R₃CH 1.4-1.7
CH₃CO- 2.0-2.5
-C≡C-H 2.5-3.0
CH₃O- 3.3-4.0
R-OH 1-5（可变）
烯烃 H 4.5-6.5
芳香 H 6.5-8.5
RCHO 9-10
RCOOH 10-13

自旋-自旋耦合

n+1 规则：被 n 个等价质子耦合，裂分为 n+1 重峰

耦合常数 J (Hz)：
  邻位（³J）：6-8 Hz
  烯烃顺式：6-12 Hz
  烯烃反式：12-18 Hz

即时检测

在¹H NMR中，乙基(-CH₂CH₃)的裂分模式是？
A. CH₂为单峰，CH₃为单峰
B. CH₂为双峰，CH₃为双峰
C. CH₂为三重峰，CH₃为二重峰
D. CH₂为四重峰，CH₃为三重峰
解析： 根据n+1规则：CH₂被相邻CH₃的3个质子耦合，裂分为3+1=4重峰；CH₃被相邻CH₂的2个质子耦合，裂分为2+1=3重峰。

第四章：质谱 (MS) ⭐⭐⭐⭐⭐

本章目标

  • 理解分子离子峰与基峰
  • 掌握氮规则
  • 能识别常见碎片离子
  • 会用同位素峰推分子式

基本原理

分子被电离 → 离子按质荷比（m/z）分离 → 检测

分子离子峰 M⁺：失去一个电子形成
基峰（Base Peak）：谱图中最强峰 = 100%

氮规则：含偶数个 N（包括 0）→ 分子量为偶数
        含奇数个 N → 分子量为奇数

常见碎片离子

碎片 m/z 来源
C₆H₅CH₂⁺ 91 烷基苯
C₆H₅CO⁺ 105 苯甲酰化合物
CH₃CO⁺ 43 甲基酮
C₂H₅⁺ 29 乙基化合物

第五章：色谱方法 ⭐⭐⭐⭐

本章目标

  • 理解van Deemter方程
  • 掌握保留时间与分离度
  • 了解HPLC与GC区别
  • 能选择合适的检测器

HPLC 高效液相色谱

分离原理：样品在固定相和流动相之间分配

保留时间 tR：从进样到峰顶的时间
容量因子 k' = (tR - t₀)/t₀

分离度 Rs = 2(tR₂-tR₁)/(W₁+W₂)
  Rs > 1.5 → 基线分离

van Deemter 方程：
  H = A + B/u + Cu
  H：理论塔板高度
  u：线速度

GC 气相色谱

适用条件：样品可气化且热稳定

常用检测器：
  FID（火焰离子化）：有机物通用，灵敏度高
  TCD（热导）：通用，灵敏度低
  ECD（电子捕获）：含卤素/硝基化合物
  MS（质谱联用）：GC-MS 是最强大的定性工具

第六章：滴定分析 ⭐⭐⭐⭐

本章目标

  • 掌握四大滴定方法
  • 理解等当点与终点
  • 能计算终点误差
  • 会做误差分析

四大滴定

类型 标准溶液 指示剂 应用
酸碱 HCl / NaOH 酚酞、甲基橙 酸/碱含量测定
配位（EDTA） EDTA 铬黑 T、二甲酚橙 金属离子测定
氧化还原 KMnO₄ / Na₂S₂O₃ 自身指示 / 淀粉 Fe²⁺、Cu²⁺ 等
沉淀 AgNO₃ K₂CrO₄（Mohr 法） 卤素离子

滴定曲线与终点判断

等当点 vs 滴定终点：
  等当点：理论反应完成点
  滴定终点：指示剂变色点

终点误差：滴定终点与等当点之间的差异
  选择合适指示剂使终点尽量接近等当点

误差分析

系统误差（可校正）：
  - 仪器误差（天平、容量瓶未校准）
  - 方法误差（副反应、共沉淀）
  - 个人误差（读数习惯）

随机误差（不可消除）：
  - 环境温度波动
  - 仪器噪声
  
精密度的衡量：
  RSD（相对标准偏差）= s/x̄ × 100%

综合谱图解析策略

1
MS：确定分子量
分子离子峰 → 氮规则 → 同位素峰 → 推测分子式
2
IR：确定官能团
特征吸收区 → 羰基/羟基/氨基等官能团判断
3
¹H NMR：氢原子环境
化学位移 → 积分面积 → 裂分模式 → 连接关系
4
¹³C NMR：碳骨架
碳原子数目 → 杂化类型 → 官能团确认
5
UV-Vis：共轭体系
λ_max → ε → 辅助确认共轭程度
未知物鉴定步骤：
1. MS → 确定分子量 → 推测分子式（氮规则 + 同位素峰）
2. IR → 确定官能团（羰基、羟基、氨基等）
3. ¹H NMR → 确定氢原子环境和连接方式
4. ¹³C NMR → 确定碳骨架
5. UV-Vis → 辅助判断共轭体系
