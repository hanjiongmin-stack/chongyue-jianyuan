第1章 绪论 ⭐⭐

▶ 有机化学发展史（选读）

1.1 有机化合物和有机化学

有机化学的定义经历了两个重要发展阶段：

  • 1848年，德国化学家葛美林提出：**有机化合物即碳化合物，有机化学是研究碳化合物的化学**
  • 1874年，德国化学家肖莱马进一步完善：**有机化合物是碳氢化合物及其衍生物，有机化学是研究碳氢化合物及其衍生物的化学**

元素分析表明，所有有机化合物均含有碳元素，绝大多数含有氢元素，许多还含有氧、氮、硫、磷及卤素等元素。

1.2 有机化学的产生和发展

有机化学的发展历程可分为三个标志性时期：

（1）萌芽时期（18世纪中叶—1858年）

  • 1828年，德国化学家维勒首次人工合成尿素，打破了"生命力论"的束缚：$\ce{NH4OCN ->[加热] (NH2)2CO}$
  • 1781年，拉瓦锡建立有机化合物的碳氢定量分析方法
  • 1830年，李比希改进碳氢分析法；同年，杜马创立氮的定量测定方法

（2）经典有机化学时期（1858年—1916年）

  • 1857年，凯库勒和库帕提出碳原子四价学说及碳链结构理论
  • 1861年，布特列洛夫提出化学结构观点，指出分子结构决定其性质
  • 1865年，凯库勒提出苯的单双键交替环状结构
  • 1874年，范特霍夫和勒贝尔建立碳的四面体学说，开创了立体化学

（3）现代有机化学时期（1916年—至今）

  • 1916年，柯塞尔和路易斯提出价键电子理论（八隅律）
  • 1931年，鲍林提出杂化轨道理论；休克尔提出芳香结构理论（休克尔规则）
  • 1952年，福井谦一提出前线轨道理论
  • 1965年，伍德沃德和霍夫曼提出分子轨道对称守恒原理

1.3 有机化合物的特点和分类

  • 结构复杂，种类繁多，同分异构现象普遍存在
  • 原子间主要通过共价键相连
  • 反应速率较慢，副反应多，产物复杂
  • 多数易燃烧，对热不稳定，熔沸点较低
  • 难溶于水，易溶于有机溶剂

1.4 有机化合物构造式的表示方法 ⭐⭐⭐

  • 1. **电子式（Lewis式）**：用圆点表示电子
  • 2. **短线式**：用短线表示共价键
  • 3. **缩简式**：省略碳氢单键，书写简便
  • 4. **键线式**：只用键线表示碳架，是有机化学最常用的表示方法

    • 第2章 共价键与有机化学反应

    本章目标

    • 理解杂化轨道理论
    • 掌握共价键参数
    • 区分均裂与异裂
    • 理解亲电与亲核试剂

    2.1 原子轨道和原子的电子构型

    碳原子的基态电子构型为 $\ce{1s^2 2s^2 2p_x^1 2p_y^1}$，通过激发，一个2s电子可被激发到空的2p轨道形成激发态，为成键提供四个未成对电子。

    2.2 共价键的形成

    （1）价键理论

    • **定域性**：成键电子定域于两个成键原子之间
    • **饱和性**：一个原子有几个未成对电子，便可与几个自旋相反的电子配对成键
    • **方向性**：原子间总是尽可能沿着原子轨道最大重叠的方向成键

    （2）杂化轨道理论

    杂化类型轨道组成空间构型键角形成的键
    $\ce{sp^3}$1个s + 3个p正四面体109.5°4个$\sigma$键
    $\ce{sp^2}$1个s + 2个p平面三角形120°3个$\sigma$键 + 1个$\pi$键
    $\ce{sp}$1个s + 1个p直线形180°2个$\sigma$键 + 2个$\pi$键

    不同杂化态碳原子的电负性：$\ce{sp C > sp^2 C > sp^3 C}$

    2.3 共价键的键参数

    • **键长**：两个原子核之间的平均距离
    • **键角**：两个共价键之间的夹角，反映分子的空间结构
    • **键能**：气态分子断裂成气态原子时所吸收的能量
    • **偶极矩**：正、负电荷中心的电荷值与距离的乘积，反映分子或键的极性

    2.4 共价键的断裂与有机化学反应 ⭐⭐⭐⭐⭐

    • **均裂**：共用电子对平均分配，产生自由基
    • **异裂**：共用电子对完全转移到一个原子上，产生离子
    • **协同反应**：旧键断裂和新键形成同时进行
      • 1
      离去基团离开
      C-X键异裂，X带走电子对离去，生成碳正离子
      R₃C-X → R₃C⁺ + X⁻
      2
      碳正离子中间体
      碳正离子为关键中间体，空间构型为平面三角形
      R₃C⁺ （sp²杂化，空p轨道）
      3
      亲核试剂进攻
      亲核试剂从平面两侧进攻碳正离子
      R₃C⁺ + Nu⁻ → R₃C-Nu
      4
      产物形成
      亲核试剂与碳正离子结合，得到外消旋产物
      R₃C-Nu （外消旋体）

      关键特征： 速率只与RX浓度有关（一级反应）；碳正离子稳定性顺序：3° > 2° > 1° > CH₃⁺；产物外消旋体。

    2.5 试剂与溶剂 ⭐⭐⭐⭐⭐

    • **亲电试剂**：电子对的受体，进攻电子云密度高的部位
    • **亲核试剂**：电子对的给体，进攻电子云密度低的部位
    • **Brönsted酸碱**：能给出质子的是酸，能接受质子的是碱
    • **Lewis酸碱**：能接受电子对的是酸，能给出电子对的是碱

    即时检测

    下列哪个化合物最容易发生SN1反应？
    A. CH₃CH₂Br
    B. CH₂=CHCH₂Br
    C. (CH₃)₃CBr
    D. CH₃Br
    解析： SN1反应速率取决于碳正离子稳定性。叔丁基溴[(CH₃)₃CBr]形成的叔丁基碳正离子最稳定（3° > 2° > 1°），因此SN1反应速率最快。

    第3章 结构与取代基效应

    3.1 有机化合物结构解析的基本方法

  • 1. 正确书写分子中原子的排序方式
  • 2. 确定官能团的类型及其成键特征
  • 3. 从电子效应、空间效应及氢键等方面分析取代基与官能团的相互影响

    • 3.2 分子间作用力

    • **范德华力**：包括取向力、诱导力和色散力
    • **氢键**：比范德华力强，具有方向性和饱和性。分子间氢键使熔沸点显著升高，分子内氢键使熔沸点降低

    3.3 取代基效应 ⭐⭐⭐⭐⭐

    （1）电子效应

    • **诱导效应（I）**：由于成键原子电负性不同引起，沿碳链传递，随距离增加迅速减弱
    • 吸电子诱导效应（-I）：电负性大于氢的原子或基团
    • 给电子诱导效应（+I）：电负性小于氢的原子或基团
    • **共轭效应（C）**：存在于共轭体系中，由于p轨道重叠使电子离域，可远程传递
    • **超共轭效应**：$\ce{C-H \sigma}$ 键与相邻的 $\pi$ 键或p轨道发生部分重叠

    （2）空间效应

    • **空间位阻**：较大基团阻碍进攻试剂与反应中心的接近
    • **空间张力**：由于立体因素使化学键处于紧张状态

    第4章 同分异构现象

    4.1 构造异构

    • **碳架异构**：碳骨架不同
    • **官能团位置异构**：官能团在碳链上的位置不同
    • **官能团异构**：官能团种类不同
    • **互变异构**：两种异构体迅速相互转化，处于动态平衡

    4.2 构象异构

    构象是由于单键旋转而产生的分子中原子或基团在空间的不同排列方式。

    典型化合物的构象：

    • **乙烷**：有交叉式和重叠式两种极限构象，交叉式更稳定
    • **正丁烷**：对位交叉构象最稳定
    • **环己烷**：椅式构象最稳定，C-H键分为直立键（a键）和平伏键（e键），取代基位于e键时更稳定

    4.3 几何异构（顺反异构）

    • **顺/反命名法**：相同基团在双键同侧为顺式，异侧为反式
    • **Z/E命名法**：优先基团在双键同侧为Z构型，异侧为E构型

    4.4 对映异构 ⭐⭐⭐⭐⭐

    • **手性碳原子**：连有四个不同原子或基团的碳原子，常用 $\ce{C^*}$ 表示
    • **R/S标记法**：将最小基团置于最远位置，其余三个基团从大到小顺时针为R，逆时针为S
    • **外消旋体**：等量左旋体和右旋体的混合物，无旋光性
    • **内消旋体**：分子内含多个手性中心但整体无旋光性

    第5章 结构表征

    5.1 紫外吸收光谱（UV）

    主要由电子跃迁产生，用于推断不饱和基团的共轭关系：

    • $\pi \to \pi^*$ 跃迁：强吸收，共轭体系增大发生红移
    • $n \to \pi^*$ 跃迁：近紫外区弱吸收（$\ce{C=O}$、$\ce{C#N}$等）

    5.2 红外吸收光谱（IR）

    • **特征频率区（4000~1500 cm⁻¹）**：官能团的伸缩振动吸收
    • **指纹区（1400~650 cm⁻¹）**：各种弯曲振动吸收，用于化合物鉴定

    5.3 核磁共振谱（NMR） ⭐⭐⭐⭐⭐

    ¹H NMR

    • **化学位移**：以TMS为标准，不同化学环境的质子共振位置不同
    • **自旋裂分**：相邻磁核之间的相互作用使谱峰分裂，遵循n+1规律
    • **偶合常数**：裂分峰之间的距离，与外加磁场强度无关
    • **峰面积**：与产生吸收的质子数目成正比

    ¹³C NMR

    • 化学位移范围广（0~250 ppm），分辨率高
    • 通常采用去偶技术，所有磁不等价的¹³C核以单峰出现

    5.4 质谱（MS）

    • **分子离子峰**：质荷比等于化合物的相对分子质量
    • **碎片离子峰**：分子离子进一步裂解产生，用于推断结构
    • **同位素峰**：由于同位素的存在而产生，可用于确定分子式

    第10章 炔烃

    本章目标

    • 掌握炔烃的结构特征
    • 理解炔氢的酸性反应
    • 掌握加成与氧化还原
    • 能设计炔烃制备路线

    10.1 炔烃的结构

    碳碳三键由一个 $\sigma$ 键和两个相互垂直的 $\pi$ 键组成，三键碳原子为 $\ce{sp}$ 杂化，分子呈直线形结构，键角180°。三键键长为120 pm，键能为837 kJ/mol。

    10.2 炔氢的酸性反应 ⭐⭐⭐⭐⭐

    末端炔烃（$\ce{RC#CH}$）的C-H键具有弱酸性，pKa≈25。可与强碱反应生成炔化物，用于增长碳链：

    $$\ce{RC#CH + NaNH2 ->[液氨] RC#CNa + NH3}$$

    $$\ce{RC#CNa + R'X -> RC#CR' + NaX \quad (R'为1°烷基)}$$

    $$\ce{RC#CH + [Ag(NH3)2]+ -> RC#CAg↓(白色) + NH4+ + NH3}$$
    $$\ce{RC#CH + [Cu(NH3)2]+ -> RC#CCu↓(红棕色) + NH4+ + NH3}$$

    $$\ce{RC#CH + [Ag(NH3)2]+ -> RC#CAg↓(白色) + NH4+ + NH3}$$

    $$\ce{RC#CH + [Cu(NH3)2]+ -> RC#CCu↓(红棕色) + NH4+ + NH3}$$

    10.3 加成反应 ⭐⭐⭐⭐⭐

    （1）亲电加成

    • 与卤素加成（反式，可停留在二卤代烯烃）：

      • $$\ce{RC#CR' + X2 -> RCX=CR'X -> RCX2-CR'X2}$$

    • 与氢卤酸加成（马氏规则，反式为主）：

      • $$\ce{RC#CH + HX -> RCX=CH2 -> RCX2-CH3}$$

    • 与水加成（库切罗夫反应，马氏规则，烯醇互变）：

      • $$\ce{HC#CH + H2O ->[HgSO4, H2SO4] CH2=CHOH <=> CH3CHO}$$

      $$\ce{RC#CH + H2O ->[HgSO4, H2SO4] RCOCH3}$$

    • 硼氢化反应：

    $$\ce{RC#CH ->[1. B2H6][2. H2O2, OH-] RCH2CHO}$$（端炔→醛）

    $$\ce{RC#CH ->[1. B2H6][2. CH3COOH] RCH=CH2}$$（顺式烯烃）

    （2）自由基加成

    $$\ce{RC#CH + HBr ->[过氧化物] RCH=CHBr}$$（反马氏规则）

    （3）亲核加成

    $$\ce{HC#CH + ROH ->[KOH] CH2=CHOR}$$

    10.4 氧化反应

    $$\ce{RC#CR' ->[KMnO4, H+] RCOOH + R'COOH}$$

    $$\ce{RC#CR' ->[1. O3][2. H2O] RCOOH + R'COOH}$$

    10.5 还原反应

    • 催化氢化：$\ce{RC#CR' + 2H2 ->[Ni/Pt/Pd] RCH2CH2R'}$
    • Lindlar催化剂：$\ce{RC#CR' + H2 ->[Lindlar催化剂] RCH=CHR'}$（顺式烯烃）
    • 钠/液氨还原：$\ce{RC#CR' + 2Na ->[液氨] RCH=CHR'}$（反式烯烃）

    10.6 聚合反应

    • 二聚：$\ce{2HC#CH ->[Cu2Cl2, NH4Cl] CH2=CH-C#CH}$
    • 三聚：$\ce{3HC#CH ->[过渡金属催化剂] C6H6}$

    10.7 制备反应

    • 二元卤代烷脱卤化氢：$\ce{RCHX-CH2X ->[KOH, 醇, \Delta] RC#CH}$
    • 炔烃烷基化（见炔氢酸性反应）

    第11章 芳烃

    本章目标

    • 理解苯的结构与Hückel规则
    • 掌握亲电取代反应类型
    • 理解定位效应规律
    • 掌握侧链氧化与Birch还原

    11.1 苯的结构

    苯分子为平面正六边形，六个碳原子均为 $\ce{sp^2}$ 杂化，p轨道侧面重叠形成闭合共轭 $\pi$ 键。C-C键长均为140 pm，共振能约150 kJ/mol。

    11.2 Hückel规则

    芳香性条件：环状闭合共轭体系 + 所有原子共平面 + $\pi$ 电子数 = $4n+2$

    11.3 苯环的亲电取代反应 ⭐⭐⭐⭐⭐

    （1）卤代反应

    $$\ce{C6H6 + X2 ->[FeX3] C6H5X + HX \quad (X=Cl, Br)}$$

    （2）硝化反应

    $$\ce{C6H6 + HNO3 ->[浓H2SO4, 50~60℃] C6H5NO2 + H2O}$$

    （3）磺化反应（可逆）

    $$\ce{C6H6 + H2SO4(浓) <=> C6H5SO3H + H2O}$$

    （4）傅-克烷基化反应（碳正离子重排）

    $$\ce{C6H6 + RX ->[AlCl3] C6H5R + HX \quad (R=烷基, X=Cl, Br)}$$

    $$\ce{C6H6 + RCH=CH2 ->[H2SO4] C6H5CH(CH3)R}$$

    （5）傅-克酰基化反应（无重排）

    $$\ce{C6H6 + RCOCl ->[AlCl3] C6H5COR + HCl}$$

    （6）氯甲基化反应

    $$\ce{C6H6 + HCHO + HCl ->[ZnCl2, \Delta] C6H5CH2Cl + H2O}$$

    （7）Gattermann-Koch 甲酰化反应

    $$\ce{C6H6 + CO + HCl ->[AlCl3, CuCl] C6H5CHO}$$

    11.4 取代基的定位效应 ⭐⭐⭐⭐⭐

    • **第一类（邻对位定位基）**：$\ce{-OH, -OR, -NH2, -NHR, -NR2, -R, -X}$（卤素为弱钝化）
    • **第二类（间位定位基）**：$\ce{-NO2, -CN, -COOH, -SO3H, -CHO, -COR}$

    即时检测

    下列哪个是间位定位基？
    A. -NO₂
    B. -OH
    C. -CH₃
    D. -NH₂
    解析： 羡基(-NO₂)是典型的间位定位基。-OH、-CH₃、-NH₂均为邻对位定位基。记住：第二类定位基的共同特征是与苯环直接相连的原子带正电荷或部分正电荷。

    11.5 加成与还原反应

    • 催化加氢：$\ce{C6H6 + 3H2 ->[Ni, 180~250℃] C6H12}$
    • 加氯（自由基）：$\ce{C6H6 + 3Cl2 ->[紫外光] C6H6Cl6}$（六六六）
    • Birch还原：$\ce{C6H6 + 2Na + 2C2H5OH ->[液氨] 1,4-环己二烯}$

    11.6 氧化反应

    • 苯环氧化：$\ce{2C6H6 + 9O2 ->[V2O5, 400~500℃] 2C4H2O3 + 4CO2 + 4H2O}$（顺丁烯二酸酐）
    • 烷基苯侧链氧化（有α-H时）：$\ce{C6H5CH2R ->[KMnO4, H+] C6H5COOH}$

    11.7 苄位氢的反应

    $$\ce{C6H5CH3 + Cl2 ->[光照] C6H5CH2Cl + HCl}$$

    11.8 萘的主要反应

    • 亲电取代（α位为主，磺化高温转β位）：

      • $$\ce{C10H8 + HNO3 ->[H2SO4] \alpha-C10H7NO2}$$

      $$\ce{C10H8 + H2SO4 ->[80℃] \alpha-C10H7SO3H ->[165℃] \beta-C10H7SO3H}$$

    • 氧化：$\ce{C10H8 ->[V2O5, 450℃] C8H4O3}$（邻苯二甲酸酐）

    第12章 醇、酚和醚

    本章目标

    • 掌握醇的亲核取代与消除
    • 理解酚的酸性与亲电取代
    • 掌握醚键断裂与环氧乙烷开环
    • 能设计Williamson醚合成

    12.1 醇 ⭐⭐⭐⭐⭐

    （1）酸性反应

    $$\ce{2ROH + 2Na -> 2RONa + H2↑}$$

    （2）亲核取代反应

    • 与氢卤酸反应：$\ce{ROH + HX -> RX + H2O}$（活性：HI>HBr>HCl；叔醇>仲醇>伯醇）
    • 与卤化磷/亚硫酰氯反应：

      • $$\ce{3ROH + PX3 -> 3RX + H3PO3}$$

    $$\ce{ROH + SOCl2 ->[吡啶] RCl + SO2↑ + HCl↑}$$（构型保持）

    • 分子间脱水成醚：$\ce{2RCH2OH ->[浓H2SO4, 140℃] RCH2OCH2R + H2O}$

    （3）消除反应（札依采夫规则）

    $$\ce{RCH2CH(OH)R' ->[浓H2SO4, \Delta] RCH=CHR' + H2O}$$

    （4）氧化与脱氢

    • 强氧化剂：$\ce{RCH2OH ->[KMnO4, H+] RCOOH}$；$\ce{R2CHOH ->[KMnO4, H+] R2C=O}$
    • 选择性氧化：$\ce{RCH=CHCH2OH ->[MnO2] RCH=CHCHO}$；$\ce{RCH2OH ->[CrO3-吡啶] RCHO}$

    （5）多元醇反应

    • 与高碘酸反应：$\ce{RCH(OH)-CH(OH)R' + HIO4 -> RCHO + R'CHO + HIO3 + H2O}$
    • 片呐醇重排：$\ce{(CH3)2C(OH)-C(OH)(CH3)2 ->[H+] (CH3)3CCOCH3}$

    12.2 酚

    （1）酸性反应

    $$\ce{C6H5OH + NaOH -> C6H5ONa + H2O}$$

    $$\ce{C6H5ONa + CO2 + H2O -> C6H5OH + NaHCO3}$$（酸性：碳酸 > 酚 > 醇）

    （2）成醚/成酯反应

    $$\ce{C6H5ONa + R'X -> C6H5OR' + NaX}$$

    $$\ce{C6H5OH + (CH3CO)2O -> C6H5OCOCH3 + CH3COOH}$$

    （3）与FeCl3显色反应

    $$\ce{6C6H5OH + FeCl3 -> H3[Fe(OC6H5)6] + 3HCl}$$（紫色，烯醇式结构特征）

    （4）芳环亲电取代

    $$\ce{C6H5OH + 3Br2 -> C6H2Br3OH↓(白色) + 3HBr}$$

    12.3 醚

    （1）𨦡盐生成：$\ce{ROR' + H2SO4(浓) -> [ROR'H]+ + HSO4-}$

    （2）醚键断裂

    $$\ce{ROR' + HI -> RI + R'OH}$$（小烷基生成碘代烷）

    （3）环氧乙烷开环

    • 酸催化（进攻取代多的碳）：$\ce{(CH2)2O + H2O ->[H+] HOCH2CH2OH}$
    • 碱催化（进攻取代少的碳）：$\ce{(CH2)2O + CH3ONa -> CH3OCH2CH2ONa ->[H2O] CH3OCH2CH2OH}$

    （4）威廉姆逊合成法

    $$\ce{RONa + R'X -> R-O-R' + NaX}$$

    第13章 醛和酮

    本章目标

    • 理解羰基亲核加成机理
    • 掌握α-H的反应
    • 能区分醛和酮的鉴别方法
    • 掌握Cannizzaro与缩合反应

    13.1 结构特征

    羰基碳为 $\ce{sp^2}$ 杂化，$\ce{C=O}$ 双键极化。亲核加成活性：$\ce{HCHO > RCHO > ArCHO > CH3COCH3 > RCOCH3}$

    13.2 亲核加成反应 ⭐⭐⭐⭐⭐

    （1）含碳亲核试剂

    • 与HCN加成：$\ce{RCHO + HCN ->[OH-] RCH(OH)CN}$
    • 与格氏试剂加成：

      • $$\ce{RCHO + R'MgX ->[1. 醚][2. H3O+] RCH(OH)R'}$$

      $$\ce{RCOR' + R''MgX ->[1. 醚][2. H3O+] RR'C(OH)R''}$$

    • Wittig反应：$\ce{R2C=O + Ph3P=CHR' -> R2C=CHR' + Ph3PO}$

    （2）含氧亲核试剂

    • 与水加成：$\ce{CCl3CHO + H2O -> CCl3CH(OH)2}$（稳定水合物）
    • 与醇加成（缩醛/酮，保护羰基）：$\ce{RCHO + 2R'OH ->[干HCl] RCH(OR')2 + H2O}$

    （3）含硫亲核试剂

    $$\ce{RCHO + NaHSO3(饱和) -> RCH(OH)SO3Na↓(白色)}$$

    （4）含氮亲核试剂

    • 与伯胺生成亚胺：$\ce{RCHO + R'NH2 -> RCH=NR' + H2O}$
    • 与氨的衍生物：$\ce{R2C=O + H2N-Y -> R2C=N-Y + H2O}$（Y=OH, NH2, NHPh等）

    13.3 α-H的反应 ⭐⭐⭐⭐⭐

    （1）卤代反应

    • 酸催化（可控）：$\ce{CH3COCH3 + Br2 ->[H+] CH3COCH2Br + HBr}$
    • 碱催化（卤仿反应，鉴别甲基酮/甲基醇）：

      • $$\ce{CH3COCH3 + 3I2 + 4NaOH -> CHI3↓(黄色) + CH3COONa + 3NaI + 3H2O}$$

    （2）羟醛缩合反应

    • 自身缩合：$\ce{2CH3CHO ->[稀NaOH] CH3CH(OH)CH2CHO ->[\Delta] CH3CH=CHCHO}$
    • 交叉缩合（Claisen-Schmidt）：$\ce{C6H5CHO + CH3COCH3 ->[稀NaOH] C6H5CH=CHCOCH3}$

    13.4 氧化还原反应

    （1）氧化反应

    • Tollens试剂（银镜反应）：$\ce{RCHO + [Ag(NH3)2]+ + OH- -> RCOONH4 + Ag↓ + NH3 + H2O}$
    • Fehling试剂：$\ce{RCHO + Cu(OH)2 + NaOH -> RCOONa + Cu2O↓(砖红色) + H2O}$
    • Cannizzaro反应（无α-H的醛）：$\ce{2HCHO + NaOH(浓) -> HCOONa + CH3OH}$
    • Baeyer-Villiger重排：$\ce{RCOR' + R''COOOH -> RCOOR' + R''COOH}$

    （2）还原反应

    • 还原为醇：$\ce{RCHO + H2 ->[Ni] RCH2OH}$；$\ce{R2C=O + NaBH4 -> R2CHOH}$
    • Clemmensen还原：$\ce{R2C=O ->[Zn-Hg, 浓HCl, \Delta] R2CH2}$
    • 黄鸣龙还原：$\ce{R2C=O ->[1. NH2NH2, NaOH][2. 缩乙二醇, \Delta] R2CH2}$

    第14章 羧酸衍生物

    本章目标

    • 掌握羧酸衍生物活性顺序
    • 理解加成-消除机理
    • 掌握Claisen缩合与Hofmann降级
    • 能设计乙酰乙酸乙酯合成

    14.1 分类与活性

    羧酸衍生物包括酰卤、酸酐、酯和酰胺。共同反应是亲核取代（加成-消除机理）。

    反应活性顺序：$\ce{RCOCl > (RCO)2O > RCOOR' > RCONH2}$

    14.2 亲核取代反应 ⭐⭐⭐⭐⭐

    （1）水解反应

    $$\ce{RCOCl + H2O -> RCOOH + HCl}$$

    $$\ce{(RCO)2O + H2O -> 2RCOOH}$$

    $$\ce{RCOOR' + H2O ->[H+/OH-] RCOOH + R'OH}$$

    $$\ce{RCONH2 + H2O ->[H+/OH-, \Delta] RCOOH + NH3}$$

    （2）醇解反应

    $$\ce{RCOCl + R'OH -> RCOOR' + HCl}$$

    $$\ce{(RCO)2O + R'OH -> RCOOR' + RCOOH}$$

    $$\ce{RCOOR'' + R'OH ->[H+/OH-] RCOOR' + R''OH}$$（酯交换）

    （3）胺解反应

    $$\ce{RCOCl + NH3 -> RCONH2 + NH4Cl}$$

    $$\ce{(RCO)2O + NH3 -> RCONH2 + RCOONH4}$$

    $$\ce{RCOOR' + NH3 -> RCONH2 + R'OH}$$

    （4）与格氏试剂反应

    $$\ce{RCOCl + 2R'MgX ->[1. 醚][2. H3O+] R-C(OH)R'2}$$

    $$\ce{RCOOR'' + 2R'MgX ->[1. 醚][2. H3O+] R-C(OH)R'2}$$

    14.3 还原反应

    • 还原为醇：$\ce{RCOCl ->[LiAlH4] RCH2OH}$；$\ce{RCOOR' ->[LiAlH4] RCH2OH + R'OH}$
    • Rosenmund还原（酰卤→醛）：$\ce{RCOCl + H2 ->[Pd-BaSO4, 硫/喹啉] RCHO + HCl}$
    • 酰胺还原为胺：$\ce{RCONH2 ->[LiAlH4] RCH2NH2}$

    14.4 特殊反应

    • **Claisen缩合**：$\ce{2CH3COOC2H5 ->[C2H5ONa] CH3COCH2COOC2H5 + C2H5OH}$
    • **Hofmann降级**：$\ce{RCONH2 + Br2 + 4NaOH -> RNH2 + Na2CO3 + 2NaBr + 2H2O}$
    • **酰胺脱水生成腈**：$\ce{RCONH2 ->[P2O5/SOCl2, \Delta] RC#N + H2O}$

    14.5 β-二羰基化合物

    • 乙酰乙酸乙酯酮式分解：$\ce{CH3COCH2COOC2H5 ->[稀NaOH, \Delta] CH3COCH3 + CO2↑ + C2H5OH}$
    • 丙二酸二乙酯合成：$\ce{CH2(COOC2H5)2 ->[1. C2H5ONa][2. RX] RCH(COOC2H5)2 ->[1. NaOH, \Delta][2. H+] RCH2COOH}$

    14.6 腈的反应

    • 水解：$\ce{RC#N + 2H2O ->[H+/OH-, \Delta] RCOOH + NH3}$
    • 还原：$\ce{RC#N ->[LiAlH4] RCH2NH2}$
